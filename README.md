### 3. Payroll Application
🔗 https://github.com/DavidDanielx72/PayrollApplication

A Java-based payroll system developed to automate employee and payroll management for a growing company.

#### Project Overview:
This system was designed for a company called STARTUP, which has over 2000 employees. The goal was to replace a manual payroll system with an automated solution to improve efficiency and provide better HR insights.

The system manages:
- Employee details (employee number, name, employment type, nationality)
- Contact details (cell phone, email, home number)
- Address details (street and postal address)
- Employee identity types (e.g., Passport)
- Job roles and positions
- Position status (OPEN or CLOSED)

The application follows a layered architecture:
- Presentation Layer
- Service Layer
- Repository Layer

#### Technologies Used:
- Java
- Object-Oriented Programming (OOP)
- IntelliJ IDEA
